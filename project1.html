Phyllotaxis Interactive Sandbox

What is a phyllotaxis?

It is an arrangement of leaves on a plant stem. From there is spirals around of patterns in nature. The term of 'phyllotaxis' was termed by Charles Bonnet to describe the arrangement of leaves on a plant.

The number of spirals can be determined by the phyllotactic pattern through the Fibonacci sequence. This angle is 137.5°, also known as the divergence angle. The formula used for the phyllotaxis is the Vogel's formula. Each new floret is at a fixed angle with respect to the preceding floret. The position for a new floret fits to the largest gap bewteen the position vectors of older florets.

Instructions on Functions

There are a few different functions to use. There are radio buttons, slider, dropdown, and button. The following are the names and functions:

  • Pre-set Degrees: Already set values to change the divergence. (radio)
  • Degrees Slider: A value range of 1 to 200, with a preset value of 137 divergence. (slider)
  • FPS: Changes the frames persecond to set values. (radio)
  • Change Background: Can choose black, off white, or gray for background. However this will restart the program. (dropdown)
  • Color: Can choose bewteen a few choices to change the color of the phyllotaxis. (dropdown)
  • Restart Canvas: Will restart the canvas with the current settings. (button)
  • Play and Pause Button: The phyllotaxis will not start until "Play" is clicked. (button)
